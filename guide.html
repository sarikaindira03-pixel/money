Personal Finance Engine

Your money,
color-coded.

Vault is a budgeting app where every spending category has a personality. Money lives in a Main Vault and routes into smart envelopes — each one behaving differently when you overspend or save.

Overview
One vault. Many personalities.
The dashboard surfaces your Main Vault balance at all times — your true financial safety net — alongside live bucket progress for every category.
Transactions
Every expense knows
where it came from.
Each transaction is tagged with its envelope color. The app shows live routing feedback — whether the spend pulled from the Main Vault, drew from the Blue Vault, or stayed quietly in your pocket.
The core idea
Five colors.
Five behaviors.
Most budget apps treat every category the same. Vault doesn't. Each envelope color defines a distinct financial personality — how it handles savings, overspending, and month-end rollover.
Orange
Reactive
No pre-set budget. Pulls directly from the Main Vault as you spend. Zero pre-planning required — built for truly unexpected costs.
Yellow
Fluid
Set a budget. Overspend? Main Vault covers it. Underspend? Savings sweep back to Main Vault at month end. Fully bidirectional.
Red
Pocket money
Underspend stays yours — the app never reclaims it. Overspend pulls from Main Vault. One of two pocket money colors.
Green
Investment
Identical logic to Red. A Investment point of view color base so you can track two distinct types of discretionary spending separately.
Blue
Isolated goal
Self-contained sinking fund. Surplus builds a private Blue Vault. Overspend draws from it only. Hard-blocked if the Blue Vault runs dry — the Main Vault is never touched.
Engineering decisions
Built to be
financially precise.
Key architectural choices that make the app reliable, not just functional.
🔒
Immutable history
Past months are locked by default. Historical vault balances can never drift — every figure is a reliable audit trail.
⛔
Blue Vault hard stop
Blue bucket overspend is blocked entirely if the Blue Vault is empty. No partial writes. No silent Main Vault drain.
♻️
Month-end sweep logic
Yellow surpluses auto-return to Main Vault. Blue surpluses accumulate privately. Each color has deterministic close-of-month behavior.
👁️
Always-visible safety net
The Main Vault balance is surfaced prominently at all times — so you always know your real financial floor, not just per-category numbers.
